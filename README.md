## Get started

1. Create a project

   ```bash
   #yarn
   yarn create expo-app --template zctechs-expo-template

   #npm
   npx create-expo-app@latest --template zctechs-expo-template
   ```

2. If you choose yarn

   ```bash
    #project directory
    touch yarn.lock && yarn
   ```

## Libraries used
